# Caso Arroyo Guerrero

## Caso real de aplicación del Kit de Verificación Comunitaria Escazú

El caso Arroyo Guerrero constituye la experiencia real a partir de la cual se desarrolló este kit.

El proceso permitió ordenar documentos oficiales, comparar respuestas institucionales, identificar contradicciones verificables, solicitar aclaraciones y construir una ruta comunitaria de defensa territorial basada en información, participación, justicia ambiental y protección de personas defensoras.

## Qué muestra este caso

Este caso muestra que una comunidad puede usar herramientas documentales sencillas para

1. Organizar respuestas oficiales
2. Comparar documentos incompatibles
3. Identificar vacíos de información ambiental
4. Solicitar aclaraciones institucionales
5. Construir trazabilidad pública
6. Evitar acusaciones sin prueba
7. Proteger la palabra comunitaria
8. Fortalecer su posición bajo el Acuerdo de Escazú

## Qué no contiene esta carpeta

Esta carpeta no contiene expedientes sensibles.

No contiene firmas, teléfonos, direcciones, rutas, amenazas, datos personales ni información que pueda aumentar riesgos para comunidades o personas defensoras.

## Resultado metodológico

El caso permitió transformar una experiencia concreta de defensa territorial en una herramienta replicable para otras comunidades.

El repositorio no busca sustituir asesoría jurídica ni decisiones comunitarias.

Busca ofrecer un método abierto para verificar documentos, hacer preguntas y fortalecer la participación pública ambiental.

# Tabla de cruce documental

Esta tabla sirve para comparar varios documentos frente a las nueve preguntas clave del kit.

## Preguntas

1. Ubicación justificada
2. Estudios ambientales suficientes
3. Licencia ambiental o concepto equivalente
4. Plan de manejo ambiental
5. Permisos ambientales específicos
6. Participación pública informada
7. Consulta previa cuando aplique
8. Viabilidad aprobada
9. Autoridad ambiental clara

## Estados posibles

SÍ

El documento dice que existe o que está hecho.

NO CONSTA

El documento dice que no existe, no fue radicado o no hay trámite.

PENDIENTE

El documento reconoce que falta actualizar, aprobar, evaluar o decidir.

?

El documento no menciona el tema o la información no es clara.

N/A

No aplica, siempre que exista una razón documentada.

## Reglas de lectura

Contradicción documental

Si un documento dice SÍ y otro documento dice NO CONSTA, PENDIENTE o algo incompatible sobre la misma pregunta, existe una alerta de contradicción documental que requiere aclaración institucional.

Ausencia de información

Si todos los documentos tienen ? sobre un tema relevante, existe una alerta de información insuficiente. No se afirma ocultamiento sin prueba. Se solicita aclaración.

Alerta comunitaria

Si aparece una contradicción, ausencia o requisito pendiente, el proyecto requiere verificación adicional y no debe presentarse comunitariamente como plenamente claro en ese punto.

Alerta reforzada de posible incumplimiento de Escazú

Si se encuentran varias contradicciones, ausencias o requisitos pendientes sobre información, participación, justicia o protección, se puede registrar una alerta reforzada para que la autoridad competente aclare, corrija o revise.

Expediente comunitario

Guardar copia de todo. La prueba pública comunitaria no es una sentencia. Es una base documentada para pedir información, participación, control institucional, protección o apoyo jurídico.
